Face Morphing

I. Project Overview

This project takes the dataset of the class photos and does face morphing between my image and another. It also calculates the mean face of the class and does some fun stuff with face morphing.

II. Methods

  • Delaunay Triangulation

    The coresponding points of different images are connected and formed into triangles using Delaunay triangulation.

  • Cross Dissolve

    Images are blended together using cross dissolving.
